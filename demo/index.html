🎮 GameLoader.js Demo

A plug-and-play gamified loading screen. The integrating developer picks one game via config; users play it while content loads.

Integration

<script src="dist/gameloader.umd.min.js"></script>
<script>
  const loader = GameLoader.init({ game: 'dino' });
  loader.show();
  loader.setProgress(40);   // optional
  loader.hide();            // when your content is ready
</script>
